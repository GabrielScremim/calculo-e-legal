Interpolação Polinomial

Bem vindos a minha primeira página no hexo!
Nessa primeira aula de interpolação linear, fizemos um exercício na folha e depois tivemos que criar um codigo, para resolver o mesmo problema. O código que crei foi feito em python na plataforma replit. A seguir estarei disponibilizando o codigo e uma foto do exercício feito em aula

Código em python

def interp_linear(x0, y0, x1, y1, x2, y2, x):
# Calcula as inclinações das retas
m1 = (y1 - y0) / (x1 - x0)
m2 = (y2 - y1) / (x2 - x1)

# Calcula os valores interpolados de y usando a equação da reta

if x >= x1: # Interpola entre x1 e x2
y = y1 + m2 * (x - x1)
else: # Interpola entre x0 e x1
y = y0 + m1 * (x - x0)
return y

# Dados da tabela

x0, y0 = 0.1, 1.221
x1, y1 = 0.6, 3.320
x2, y2 = 0.8, 4.953
x = 0.2

# Calcula P(0.2) usando interpolação linear

resultado = interp_linear(x0, y0, x1, y1, x2, y2, x)
print(“P(0.2) =”, resultado)
